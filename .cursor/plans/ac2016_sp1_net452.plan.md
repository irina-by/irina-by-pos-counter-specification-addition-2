# AutoCAD 2016 SP1 (M.49.0) — совместимость net452

## Контекст

| Параметр | Значение |
|----------|----------|
| Продукт | AutoCAD 2016 SP1 |
| Ядро | R20.1 SP1 |
| Сборка | M.49.0 |
| .NET в AutoCAD | **4.5.2** (после SP1) |

Раньше проект собирался как **net46** (.NET 4.6+) — на AC 2016 SP1 с CLR 4.5.2 NETLOAD мог требовать установки .NET 4.6.

## Решение

Целевой framework **net452** (.NET Framework 4.5.2):
- загружается в AutoCAD 2016 SP1 без доп. runtime;
- работает на AC 2017–2024 (CLR выше 4.5.2);
- `System.ValueTuple` — `lib\netstandard1.0`.

Папка деплоя по-прежнему `dll 2016\` (имя для инженера).

## Проверка

1. `build\build-ac2016.cmd`
2. NETLOAD → CMD: `[POSC] … (net452) загружен`
3. ЗАПУСТИТЬ → Выбрать спецификацию
